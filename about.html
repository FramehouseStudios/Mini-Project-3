What if Blockbuster survived?

Blockbuster+ imagines the video store evolving into a premium movie discovery platform: nostalgic like Friday night rentals, polished like modern streaming, and social enough to feel curated by real film fans.

The Concept

The app combines the visual memory of Blockbuster shelves, VHS texture, and yellow-blue branding with the speed and layout of a streaming catalog. Users can explore films, search by title, filter by genre, and add rentals to a queue.

Technologies Used

  • HTML5 semantic page structure
  • CSS gradients, Grid, Flexbox, animation, and responsive design
  • JavaScript with async/await, fetch(), filter(), and forEach()
  • Bootstrap 5 navbar, buttons, forms, spacing, and grid system
  • Local JSON data for movie catalog content

Design Inspiration

The interface borrows Blockbuster's bright yellow confidence, Netflix's browsable catalog rhythm, Spotify's premium dark UI, and Letterboxd's film-discovery energy. The scanline overlay and glowing type add a VHS-era signal without making the app feel old.

Future Improvements

  • Save the queue with localStorage
  • Add user ratings and review cards
  • Create actor, decade, and mood filters
  • Add trailers, watchlist sorting, and rental history
  • Deploy the app with GitHub Pages or Netlify
